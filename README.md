# hashnode-userscripts

A small collection of userscripts that improve the Hashnode experience.

## Included scripts

- **hashnode-qol-enhancer** — Adds small quality-of-life features for Hashnode writers:
	- Preview button on the Hashnode draft editor (opens a preview of the current draft on the target blog).
	- Edit button on *.hashnode.dev posts (opens the post in the Hashnode editor).
	See [hashnode-qol-enhancer](scripts/hashnode-qol-enhancer/README.md) for details and installation instructions.
